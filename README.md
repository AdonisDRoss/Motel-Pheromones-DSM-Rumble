# 🎮 Motel Pheromones

A 2D arena brawler + co-op city traversal combat system built around:

- Grid-based world progression
- Combo-driven combat system
- Groove momentum mechanic
- Faction-based enemy AI
- Tag-based Arena Battle Mode

---

## 🧠 CORE SYSTEMS

### Combat
- Frame-based attacks
- Hitbox + hurtbox system
- Combo scaling system
- Groove momentum system

### World
- 4x5 grid city (University Street, Des Moines)
- Node-based encounters
- Shops, objectives, boss gates

### Modes
- Single Player (adaptive AI world)
- Co-op (1–4 players scaling system)
- Arena Battle (TAG SYSTEM ENABLED)
- Survival Mode
- City Control Mode

---

## ⚔️ SPECIAL RULES

- Enemies do NOT use specials
- Sub-bosses: 1 special
- Bosses: 2 specials
- Tag system ONLY exists in Arena Battle Mode

---

## 🧱 SPRITE SYSTEM

- 128x128 per frame
- 16-column grid system
- Fixed row-based animation layout
- Transparent PNG only

---

## 🎵 UNIQUE MECHANIC

### Groove System
Builds combat momentum and unlocks:
- speed boosts
- combo extensions
- visual distortion effects
- “Groove Break” state

---

## 🚀 DEVELOPMENT GOAL

Create a readable, high-speed, combo-driven brawler with systemic AI behavior and co-op synergy depth.
